Automated OMR Evaluator

Fast, auditable OMR grading with overlays, CSV/JSON export and a beautiful UI.

MVP

How it works

  1. Upload OMR images (PNG/JPG) or single-page PDFs in the Streamlit UI below.
  2. The OMR pipeline detects filled bubbles and generates an overlay for audit.
  3. Download per-sheet JSON and a batch CSV for records and analysis.

Tip: Upload a JSON answer key in the sidebar to use your official keys.

Quick actions

Open the Streamlit controls on this page to upload files and run batch grading.

Results are saved in the results/ folder for auditing.

For production, replace the demo answer key and refine detection parameters.
